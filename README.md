# Tic Tac Toe Using Python

A simple and interactive GUI-based Tic Tac Toe game developed using Python and Tkinter. This project allows two players to play the classic Tic Tac Toe game on a graphical interface with automatic win and draw detection. The application is designed with a clean and user-friendly interface, making it easy to play and understand.

---
## Project Overview

The Tic Tac Toe game is a two-player strategy game where players take turns marking spaces in a 3×3 grid using symbols "X" and "O". The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

This project demonstrates the implementation of graphical user interfaces (GUI), event-driven programming, and game logic using Python.

---

## Features

- Two-player gameplay
- Interactive graphical user interface
- Automatic win detection
- Draw detection functionality
- Restart game option
- Real-time player turn updates
- Clean and responsive design

---

## Technologies Used

- Python
- Tkinter (Python GUI Library)
